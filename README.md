Esse é um miniprojeto de banco de dados em Python desenvolvido na Fatec no curso de Inteligência Artificial para a disciplina Linguagem de Programação 2.
O miniprojeto consiste em um programa Python que se conecta em um banco de dados Relacional Postgree e executa as seguintes atividades:

Atividade 1
Anonimiza os seguintes dados do banco:
Nome: Mostra a primeira letra e anonimiza com asteristicos as outras letras do primeiro nome e mostra o sobrenome sem anonimiza-lo
CPF: Mostra apenas os 3 primeiros digitos e o restante anonimiza com asteristicos
E-mail: Mostra a primeira letra, anonimiza com asteristicos o restante até o @, e do @ em diante exibe sem anonimizar
Telefone: Exibe apenas os 4 ultimos digitos

Atividade 2
Cria arquivos CSV para cada ano de nascimento, exemplo 1960.CSV só conterá pessoas do banco de dados que nasceram no ano de 1960 e assim para cada ano. (Observação: para essa atividade os dados permanecem anonimizados assim como a atividade 1).

Atividade 3
Cria um arquivo CSV com o nome completo e CPF de todos os registros do banco de dados, sem anonimização.

Atividade 4
Calcula o tempo que o programa leva para executar as atividades 2 e 3 e salva em um arquivo txt.


Para rodar o projeto instale as bibliotecas contidas no arquivo requirements.txt e execute o arquivo lgpd.py
